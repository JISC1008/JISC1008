# 🧪 Project 1 | Multi-Omics Integration to Identify Drivers of Mucosal Healing in Paediatric Crohn’s Disease

**Clinical Data · NMR-based Metabolic Phenotyping · Microbiome Analysis**

---

## 📌 Project Summary

This project presents an end-to-end **multi-omics data science workflow** integrating **clinical data, NMR-based metabolomics, and gut microbiome profiles** to identify biological features associated with **mucosal healing versus persistent inflammation** in paediatric Crohn’s disease (CD).

The analysis reproduces and translates insights from a real-world clinical research study into an **industry-grade, interpretable data science pipeline**, emphasizing **biomarker discovery and translational relevance**.

---

## 🎯 Problem Statement

> Are there microbial or metabolic features in established paediatric Crohn’s disease that contribute to **sustained mucosal inflammation** or **promote mucosal healing**?

This question is highly relevant to:
- Biotech & pharma R&D  
- Precision medicine  
- Translational data science  
- Health and life-science analytics  

---

## 📂 Data Description

This project integrates three heterogeneous data layers:

### 1️⃣ Clinical Data
- Calprotectin (healing vs active inflammation)
- Disease characteristics and patient metadata

### 2️⃣ Metabolomics
- ¹H-NMR–based metabolic phenotyping

### 3️⃣ Microbiome
- Gut microbial relative abundances
- Taxonomic profiling

> ⚠️ **Data note:**  
> Original patient-level data are not publicly available.  

---

## 🔬 Analytical Workflow

### 🧹 Data Preprocessing
- Data harmonization across modalities
- Normalization and scaling
- Handling of missing values
- Filtering of low-abundance microbial features

### 📊 Exploratory Data Analysis (EDA)
- Group comparisons (mucosal healing vs inflammation)
- Feature distributions and correlations
- Dimensionality reduction (PCA)

### 🔗 Multi-Omics Integration
- Joint analysis of metabolomic and microbiome data
- Correlation-based integration
- Identification of co-varying microbial–metabolite patterns

### 📐 Statistical Modeling
- Univariate and multivariate association testing
- Adjustment for clinical covariates
- Effect size estimation and interpretability-focused modeling

---

## 📈 Key Insights

### 🔹 Metabolic Signature of Ongoing Inflammation
- **Lysine** levels were elevated in patients with **persistent mucosal inflammation**
- Suggests altered amino-acid metabolism linked to disease activity

### 🔹 Microbial–Metabolite Axis of Mucosal Healing
- **Dialister.sp** showed increased abundance in patients with **mucosal healing**
- **Pentanoate**, a short-chain fatty acid with known immunomodulatory properties:
  - Was increased in mucosal healing
  - Showed a positive correlation with Dialister abundance

### 🔹 Translational Interpretation
- The **Dialister–pentanoate axis** represents a **novel, potentially targetable biological pathway**
- Highlights a microbiome-driven metabolic mechanism that may promote mucosal healing in CD

# Project 2 | 📊 Repeated-Measures Machine Learning for Novel Metabolite Identification in GI Data

**Clinical Data · NMR-based Metabolic Phenotyping · Microbiome Analysis**

---

## 📚 Related Publication (Author’s Contribution)

This repository reflects analytical work conducted by the author and published in a peer-reviewed study, with a specific focus on **novel metabolite discovery using longitudinal modeling**.

**Key author contribution**
- Identified **stachyose in gastrointestinal samples for the first time** using **¹H-NMR-based metabolic profiling**
- Applied **repeated-measures statistical modeling** combining:
  - Univariate analysis
  - Multivariate machine learning (dimensionality reduction, pattern discovery)
- Demonstrated biologically meaningful temporal associations across individuals

### My Role
- Designed and executed the longitudinal statistical analysis
- Performed NMR-based metabolite identification
- Integrated repeated-measures modeling with multivariate ML
- Interpreted findings in the context of diet–host interactions

This finding represented a **previously unreported dietary oligosaccharide signal** in GI samples and became a **foundational insight for subsequent projects in personalised nutrition and diet–host interaction modeling**.

## 📚 Related Publication (Author’s Work)

**Dagbasi et al.**  
*Diet shapes the metabolite profile in the intact human ileum, which affects PYY release*  
**Science translational medicine**, 2024.  

🔗 https://www.science.org/doi/10.1126/scitranslmed.adm8132

This project represents the author’s contribution to the first identification of stachyose in GI samples using NMR-based metabolic phenotyping and repeated-measures machine learning, providing a key methodological and biological insight for personalised nutrition research.
